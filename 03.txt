根据你的反馈，问题核心在于 读取电台列表时不应该依赖存储权限。即使在高版本 Android 上未授予存储权限，应用也应当能正常显示内置电台并播放。只有在用户主动增删电台时才需要请求权限并尝试写入 M3U 文件。

以下方案将实现：

1. 启动时直接加载内置默认电台列表（不读取 M3U 文件）。
2. 若曾成功保存过自定义列表到 M3U 文件，则尝试读取该文件覆盖默认列表（但读取失败时绝不阻塞，静默回退）。
3. 增删电台时才检查并请求存储权限，权限授予后写入 M3U 文件，失败则提示用户“仅本次有效”。

MainActivity.kt 仅在增删电台时请求权限，移除 onCreate 中的权限检查

二、StationStorage.kt 修改方案

核心改动说明

· getStations() 默认返回内置默认电台列表。
· 若 M3U 文件存在且可读，则尝试解析并覆盖默认列表（使用户自定义列表生效）。
· 若文件不存在或无法读取，不报错，不请求权限，直接使用内置列表。
· 增删操作返回 Boolean 表示是否成功写入文件，由 MainActivity 处理提示。

替换 getStations() 方法

```kotlin
fun getStations(): List<Station> {
    // 始终从内置默认电台开始
    val defaultStations = getDefaultStations()
    val m3uFile = getM3UFile()
    
    // 尝试读取 M3U 文件（如果存在且可读）
    return try {
        if (m3uFile.exists() && m3uFile.canRead()) {
            val parsed = parseM3U(m3uFile)
            if (parsed.isNotEmpty()) parsed else defaultStations
        } else {
            defaultStations
        }
    } catch (e: Exception) {
        // 任何读取异常均回退到内置列表
        e.printStackTrace()
        defaultStations
    }
}
```

完整 StationStorage.kt （关键部分）

```kotlin
package TV.radio.data

import android.content.Context
import android.content.SharedPreferences
import android.os.Environment
import java.io.File

class StationStorage(private val context: Context) {

    companion object {
        private const val PREFS_NAME = "tv_radio_prefs"
        private const val KEY_LAST_PLAYED_ID = "last_played_id"
        private const val KEY_LAST_VOLUME = "last_volume"
        private const val KEY_LAST_POSITION = "last_position"
        private const val KEY_USE_EXO_PLAYER = "use_exo_player"
        private const val KEY_USE_HARDWARE_DECODE = "use_hardware_decode"

        private const val M3U_DIR = "tvradio"
        private const val M3U_FILE = "radio.m3u"
    }

    private val prefs: SharedPreferences = context.getSharedPreferences(PREFS_NAME, Context.MODE_PRIVATE)

    private fun getM3UFile(): File {
        val dir = File(Environment.getExternalStorageDirectory(), M3U_DIR)
        return File(dir, M3U_FILE)
    }

    // ========== 电台列表读取（不依赖权限） ==========
    fun getStations(): List<Station> {
        val defaultStations = getDefaultStations()
        val m3uFile = getM3UFile()
        return try {
            if (m3uFile.exists() && m3uFile.canRead()) {
                val parsed = parseM3U(m3uFile)
                if (parsed.isNotEmpty()) parsed else defaultStations
            } else {
                defaultStations
            }
        } catch (e: Exception) {
            e.printStackTrace()
            defaultStations
        }
    }

    // ========== 增删改（需要权限，返回是否成功写入文件） ==========
    fun addStation(station: Station): Boolean {
        val stations = getStations().toMutableList()
        if (stations.none { it.name == station.name && it.url == station.url }) {
            stations.add(station)
            return saveStationsToM3U(stations, getM3UFile())
        }
        return false
    }

    fun removeStation(station: Station): Boolean {
        val stations = getStations().toMutableList()
        val iterator = stations.iterator()
        var removed = false
        while (iterator.hasNext()) {
            if (iterator.next().id == station.id) {
                iterator.remove()
                removed = true
                break
            }
        }
        return if (removed) {
            saveStationsToM3U(stations, getM3UFile())
        } else {
            false
        }
    }

    // 写入 M3U 文件（可能因权限不足失败）
    private fun saveStationsToM3U(stations: List<Station>, m3uFile: File): Boolean {
        return try {
            val dir = m3uFile.parentFile
            if (dir?.exists() == false) {
                dir.mkdirs()
            }
            m3uFile.bufferedWriter().use { writer ->
                writer.write("#EXTM3U\n")
                stations.forEach { station ->
                    val displayName = if (station.description.isNotBlank()) {
                        "${station.name} - ${station.description}"
                    } else {
                        station.name
                    }
                    writer.write("#EXTINF:0,$displayName\n")
                    writer.write("${station.url}\n")
                }
            }
            true
        } catch (e: Exception) {
            e.printStackTrace()
            false
        }
    }

    private fun parseM3U(m3uFile: File): List<Station> {
        val stations = mutableListOf<Station>()
        try {
            val lines = m3uFile.readLines()
            var i = 0
            while (i < lines.size) {
                var line = lines[i].trim()
                if (line.isEmpty() || line == "#EXTM3U") {
                    i++
                    continue
                }
                if (line.startsWith("#EXTINF:")) {
                    val displayName = line.substringAfter(",", "").trim()
                    var name = displayName
                    var description = ""
                    if (displayName.contains(" - ")) {
                        val parts = displayName.split(" - ", limit = 2)
                        name = parts[0].trim()
                        description = parts[1].trim()
                    }
                    i++
                    if (i < lines.size) {
                        val url = lines[i].trim()
                        if (url.isNotEmpty() && !url.startsWith("#")) {
                            stations.add(Station(name = name, url = url, description = description))
                        }
                    }
                } else {
                    val url = line
                    if (url.startsWith("http://") || url.startsWith("https://")) {
                        stations.add(Station(name = url.substringAfterLast("/", "未知电台"), url = url))
                    }
                }
                i++
            }
        } catch (e: Exception) {
            e.printStackTrace()
        }
        return stations
    }

    // ========== 内置默认电台（保持不变） ==========
    private fun getDefaultStations(): List<Station> {
        return listOf(
            Station("湖南交通广播", "http://a.live.hnradio.com/jtpd/radio120k_jtpd.m3u8?auth_key=1588751155-0-0-301d7e28868eff70a72edf5e4569b546", "湖南交通广播"),
            Station("CNR-1-中国之声", "http://ngcdn001.cnr.cn/live/zgzs/index.m3u8", "CNR-1-中国之声"),
            // ... 其余内置电台保持不变 ...
        )
    }

    // ========== 配置项（SharedPreferences 不变） ==========
    // ... 与之前相同 ...
}
```

三、MainActivity.kt 修改方案

1. 移除 onCreate 中的权限检查和请求

删除或注释掉以下代码：

```kotlin
// 检查存储权限
if (!hasStoragePermission()) {
    requestStoragePermission()
} else {
    loadStations()
    restoreLastPlayed()
}
```

直接改为：

```kotlin
loadStations()
restoreLastPlayed()
```

2. 增删电台时动态请求权限并处理

修改 addStation 和 deleteStation 方法，先检查权限，若无权限则请求，并在回调中执行实际操作。

```kotlin
private fun addStation(station: Station) {
    if (!station.isValid()) {
        Toast.makeText(this, R.string.error_invalid_station, Toast.LENGTH_SHORT).show()
        return
    }

    if (hasStoragePermission()) {
        // 已有权限，直接保存
        performAddStation(station)
    } else {
        // 无权限，请求权限
        pendingAddStation = station // 暂存待添加的电台
        requestStoragePermission()
    }
}

private fun performAddStation(station: Station) {
    val saved = stationStorage.addStation(station)
    loadStations()
    if (saved) {
        Toast.makeText(this, R.string.station_added, Toast.LENGTH_SHORT).show()
    } else {
        Toast.makeText(this, "电台已添加至列表，但因权限限制未保存到文件", Toast.LENGTH_LONG).show()
    }
}
```

类似处理删除操作。需要添加成员变量暂存操作对象：

```kotlin
private var pendingAddStation: Station? = null
private var pendingDeleteStation: Station? = null
```

3. 权限回调中执行暂存操作

```kotlin
override fun onRequestPermissionsResult(
    requestCode: Int,
    permissions: Array<out String>,
    grantResults: IntArray
) {
    super.onRequestPermissionsResult(requestCode, permissions, grantResults)
    if (requestCode == REQUEST_CODE_STORAGE) {
        if (grantResults.isNotEmpty() && grantResults[0] == PackageManager.PERMISSION_GRANTED) {
            // 权限已授予，执行暂存的操作
            pendingAddStation?.let {
                performAddStation(it)
                pendingAddStation = null
            }
            pendingDeleteStation?.let {
                performDeleteStation(it)
                pendingDeleteStation = null
            }
        } else {
            Toast.makeText(this, "未授予存储权限，电台将仅在本次运行中生效", Toast.LENGTH_LONG).show()
            // 即使无权限，也应在内存中添加/删除（保证UI一致性）
            pendingAddStation?.let {
                // 强制添加但不保存文件
                stationStorage.addStation(it) // addStation 内部会尝试写文件并返回 false，但内存列表会变
                loadStations()
                pendingAddStation = null
            }
            pendingDeleteStation?.let {
                stationStorage.removeStation(it)
                loadStations()
                pendingDeleteStation = null
            }
        }
    }
}
```

注意：由于 addStation 和 removeStation 内部都会调用 getStations() 重新读取文件，如果无权限且文件不存在，读取将回退到内置列表，导致刚才的内存添加丢失。因此我们需要调整策略：当无权限时，不应调用 addStation 的写入逻辑，而是直接操作内存列表。

更简洁的方案：在 StationStorage 中增加一个内存缓存列表，仅在应用生命周期内有效。当无权限时，直接操作该缓存列表，不尝试写文件。

四、StationStorage 增加内存缓存（推荐）

为避免无权限时增删操作被文件读取覆盖，可以在 StationStorage 中维护一个 cachedStations 变量：

```kotlin
class StationStorage(private val context: Context) {

    // 内存缓存（本次运行期间的自定义列表）
    private var cachedStations: MutableList<Station>? = null

    fun getStations(): List<Station> {
        // 优先返回缓存（如果有）
        cachedStations?.let { return it.toList() }

        val defaultStations = getDefaultStations()
        val m3uFile = getM3UFile()
        val stations = try {
            if (m3uFile.exists() && m3uFile.canRead()) {
                val parsed = parseM3U(m3uFile)
                if (parsed.isNotEmpty()) parsed else defaultStations
            } else {
                defaultStations
            }
        } catch (e: Exception) {
            defaultStations
        }
        cachedStations = stations.toMutableList()
        return stations
    }

    fun addStation(station: Station): Boolean {
        val stations = getStations().toMutableList()
        if (stations.none { it.name == station.name && it.url == station.url }) {
            stations.add(station)
            cachedStations = stations
            return saveStationsToM3U(stations, getM3UFile())
        }
        return false
    }

    fun removeStation(station: Station): Boolean {
        val stations = getStations().toMutableList()
        val iterator = stations.iterator()
        var removed = false
        while (iterator.hasNext()) {
            if (iterator.next().id == station.id) {
                iterator.remove()
                removed = true
                break
            }
        }
        if (removed) {
            cachedStations = stations
            return saveStationsToM3U(stations, getM3UFile())
        }
        return false
    }

    // 当写入文件失败时，缓存仍然有效，下次 getStations 返回缓存内容
}
```

这样即使写入文件失败，cachedStations 仍然保存了修改后的列表，下次 getStations() 会优先返回缓存，确保本次运行期间增删生效。

五、最终效果

· 启动时：无任何权限弹窗，直接显示内置电台（或之前成功保存的 M3U 列表）。
· 添加/删除电台时：
  · 若未授权存储权限，弹出系统权限请求。
  · 用户同意 → 写入 M3U 文件，持久化。
  · 用户拒绝 → 仅内存生效，提示“仅本次有效”。
· Android 4.4 兼容：由于 minSdk 19，Environment.getExternalStorageDirectory() 可用，且无需动态权限（系统自动授予），因此低版本上增删可直接写入文件，体验正常。
